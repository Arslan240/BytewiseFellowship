## Ternary Operator

Ternary operator is short form of *if else condition*.
> `condition ? true : false`
> <br> 
> //in react case <br>
> `condition ? (<Component></Component>) : (<></>)`

<br>

## Nullish Coalescing

Short form for only *if condition*. There is no else part

> `condition ?? true`
>
